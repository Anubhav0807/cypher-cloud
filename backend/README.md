# Backend of Cypher Cloud in Node.js

The .env file and add the following
```bash
DB_USERERNAME=your_mongodb_atlas_username
DB_PASSWORD=your_mongodb_atlas_password

S3_BUCKET_A_NAME=cypher-cloud-s3-bucket-a
S3_BUCKET_B_NAME=cypher-cloud-s3-bucket-b
S3_BUCKET_C_NAME=cypher-cloud-s3-bucket-c
S3_ACCESS_KEY_ID=your_access_key_id
S3_SECRET_ACCESS_KEY=your_secret_access_key
S3_REGION=ap-south-1

FILE_ENCRYPTION_KEY=your_file_encryption_key

SECRET_ACCESS_TOKEN=your_secret_access_token
SECRET_REFRESH_TOKEN=your_secret_refresh_token

MAIL_USERNAME=example@gmail.com
MAIL_PASSWORD=your_gmail_app_password

FRONTEND_URL=http://localhost:3000
```
